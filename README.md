# Timetable Generation and Conflict Resolution System

A web-based system developed using ASP.NET to automate the generation of academic timetables while resolving scheduling conflicts.

## 🚀 Features
- Automatic timetable generation
- Conflict detection and resolution
- Course, lecturer, and venue management

## 🛠️ Tech Stack
- ASP.NET (C#)
- SQL Server
- HTML/CSS/Bootstrap
